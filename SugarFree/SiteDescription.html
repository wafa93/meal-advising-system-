Project goal

The main aim is to build a meal advising system and to a build recommendation path for people to help them plan their meals, based on what they prefer and the similarities of recipes.

CSS:

In general, single external style sheet was used to ensure the exact or similar style of all pages in general. Some of file the embedded was used in some pages to further customize the formatting.

JavaScript:

On the welcome page, the site welcomes the user by returning the name stored in the database

On the site description page as a type of user interaction, the user name is required before the page is displayed. The page welcomes the username followed by the page information.
